# Hero IO

## Description
Hero IO is a React web app to browse, view, and install apps. Users can see top apps, search all apps, view details, install/uninstall apps, and manage installed apps. The app uses charts to show reviews and saves installed apps in localStorage.

## Features
- Header with logo, navigation links, and Contribution button.
- Home page with banner, states section, and top apps.
- All Apps page with search and live filtering.
- App Details page with install button, description, and review chart.
- My Installation page to manage installed apps with uninstall feature.
- Custom footer and 404 error page.
- Loading animation for navigation and search.
